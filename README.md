# JSP_Webmarket_Project (1인)
## 목적 ##
+ 신입 개발자 기본
+ 웹 사이트 개발 시 필수요소
+ 실력과 경험 쌓기
------------
# 개발 환경 
> Front-End
+ HTML
+ CSS
+ JavaScript
+ Bootstrap 4
> Back-End
+ Java 1.8
+ JSP
+ MySQL 8.0
+ Tomcat 9.0
------------
